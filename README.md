## Installation

```bash
$ git clone https://github.com/imkey-org/imkey
$ cd imkey
$ yarn install
```

### Running the app

```bash
# development
$ yarn run dev

# build
$ yarn run build

# production mode
$ yarn run start
```

### Running db sync schemas

```bash
# generate schema
$ yarn run postinstall

# run db push
$ yarn run db:push
```

### Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
